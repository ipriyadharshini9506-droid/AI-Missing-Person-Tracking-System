AI Missing Person Identification & Smart Tracking System

This project is an AI-based system that helps authorities locate missing persons quickly using:

- AI Face Recognition
- CCTV Surveillance
- Route Prediction
- Transport Monitoring
- Crowd Density Analysis
- Citizen Reporting Platform

Technologies Used:
HTML, CSS, JavaScript, React.js
Python, Flask/Django
OpenCV, TensorFlow, YOLO
MongoDB / MySQL
Google Maps API
